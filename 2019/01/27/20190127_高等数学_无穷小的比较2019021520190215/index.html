高等数学_无穷小的比较

数据分析 高等数学 2019/01/27

1.无穷小的定义

设$\alpha , \beta$ 是同一自变量变化过程中的无穷小，且 $\alpha \neq 0 , \lim \frac{\beta}{\alpha}$ 表示这一变换过程中的极限。

  1. 如果 $\lim \frac{\beta}{\alpha} = 0$,则称 $\beta$ 是比$\alpha$高阶的无穷小，记作$\beta = \omicron(\alpha)$
  2. 如果 $\lim \frac{\beta}{\alpha} = \infty$,则称 $\beta$ 是比$\alpha$低阶的无穷小
  3. 如果 $\lim \frac{\beta}{\alpha} = c \neq 0$,则称 $\beta$与$\alpha$同阶无穷小
  4. 如果 $\lim \frac{\beta}{\alpha} = 1 $,则称 $\beta$与$\alpha$等价无穷小,记作$\alpha $~$ \beta$
  5. 如果 $\lim \frac{\beta}{\alpha^k} = c \neq 0$ k>0,则称 $\beta$是关于$\alpha$的k阶无穷小

2. 等价无穷小的性质

  1. $\alpha$ ~ $\beta$的充分必要条件是$\beta = \alpha + \omicron(\alpha)$
  2. $\alpha$ ~ $\alpha^{‘}$,$\beta$ ~ $\beta^{‘}$,且$\lim \frac{\beta^{‘}}{\alpha^{‘}}$存在，则
    $\lim \frac{\beta}{\alpha}$ = $\lim \frac{\beta^{‘}}{\alpha^{‘}}$
  3. 在利用等价无穷小代换求极限时，只有对所求极限中相乘或相除的因式才能用等价无穷小来代替，而对极限中相加或相减的部分则不能随意来代替。
